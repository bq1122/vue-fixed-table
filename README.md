# vue table 组件

[Demo](https://zwingz.github.io/vue-fixed-table/release/demo.html)


功能：
+ 固定表格头、左侧栏
+ 单元格整体居中，局部对齐
+ 虚拟的横向滚动条


----
## 主要依赖

`vue@2.4.4`
----

## Build Setup

+ git clone

+ cd dir,执行

    `npm install`
    
    完成依赖包安装
    

+ 开发环境(访问 localhost:8082)

    `npm run dev`

+ 生产环境

    `npm run build`
